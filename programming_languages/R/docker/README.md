# Анализ гемоглобина - Docker контейнер

## Сборка и запуск:

```bash
# Сборка образа
docker build -t hemoglobin-analysis .

# Запуск контейнера с сохранением результатов
docker run -v $(pwd)/results:/app/results hemoglobin-analysis
```


## Подготовка файлов

```bash
# 1. Переходим в домашнюю директорию и создаем папку проекта
cd ~
mkdir r_hw
cd r_hw

# 2. Копируем файл с данными
cp /path/to/Пациенты.xlsx .
cp /path/to/data_exploration.R .
```
